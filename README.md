## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## ğŸ“‹ About LaraCommerce

LaraCommerce is a comprehensive, production-ready e-commerce platform built on Laravel 8. It provides a complete solution for online selling with both administrative and customer-facing features. The platform is designed to be scalable, secure, and easy to customize.

## âœ¨ Features

### Product Management

- **Product Catalog**: Complete product management with detailed product information
- **Multiple Product Images**: Support for multiple product images with featured image selection
- **Product Attributes**: Flexible product attributes with customizable values
- **Product Variants**: Support for product attributes and variations
- **Stock Management**: Real-time inventory tracking with current and total stock counts
- **Product Pricing**: Regular price, sale price, and purchase price tracking

## ğŸš€ Installation

### 1. Clone the Repository

```bash
git clone <repository-url>
cd laracommerce
```

### 2. Install PHP Dependencies

```bash
composer install
```

### 3. Install Node.js Dependencies

```bash
npm install
```

### 4. Environment Configuration

```bash
cp .env.example .env
php artisan key:generate
```

Update the `.env` file with your database and SSLCommerz credentials:

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laracommerce
DB_USERNAME=root
DB_PASSWORD=

APP_NAME="LaraCommerce"
```

### 5. Database Setup

```bash
php artisan migrate
php artisan db:seed
```

### 6. Build Assets

```bash
# Development
npm run dev

# Production
npm run production

# Watch for changes
npm run watch
```

### 7. Start the Server

```bash
php artisan serve
```

The application will be available at `http://localhost:8000`
